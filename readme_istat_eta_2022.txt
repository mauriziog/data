Il file istat_eta_2022.csv contiene i seguenti campi:

eta - numerico
maschi - numerico
femmine - numerico
totale - numerico

i dati sono riferiti alla popolaziuone italiana al 01.01.2022 e sono disponibili a questo indirizzo: https://www.istat.it/it/archivio/269158 
Da questo file sono state estratte le colonne ed aggiunto all'età "100" tutta la popolazione anche con ètà maggiore di 100.

Il programma per provare questo dataset è prova_data_2.py
